# Auth System - SpecKit Test Project

This project is a test of the SpecKit workflow for generating a complete user authentication system.

## Project Description

User authentication system with:
- Email/password authentication
- Social login (Google, GitHub)
- Password reset flow
- Session management
- Protected routes

## SpecKit Workflow Test

This project will be used to test the complete SpecKit pipeline:
1. `/speckit-constitution` - Define principles and standards
2. `/speckit-specify` - Generate functional requirements
3. `/speckit-plan` - Create technical architecture
4. `/speckit-tasks` - Generate implementation stories
5. `/speckit-convert` - Convert to prd.json for Ralph++
6. `/ralph-loop` - Autonomous implementation
