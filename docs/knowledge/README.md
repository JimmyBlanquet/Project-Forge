## Comment Contribuer

Quand ajouter un article à la knowledge base ?

**✅ Ajouter quand :**
- Découverte d'un pattern réutilisable
- Solution à problème non-trivial
- Best practice validée sur plusieurs cas
- Trick/tip utile avec un outil

**❌ Ne PAS ajouter quand :**
- Information déjà dans documentation officielle
- Trop spécifique à un cas unique
- Évident/basique
